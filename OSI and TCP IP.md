# 協定protocol
communcation protocol 通訊協定
為什麼要分層?
把通訊協定的某些特定功能獨立起來執行
# OSI:
7  app  給應用軟體用的介面
6  presentation  雙方互相溝通用的格式
5  session  負責維護兩台電腦之間的通訊連接
4  transport  TCP/UDP等網路協定
3  network  網際網路協定
2  data link  實體位址
1  physical  負責管理電腦通訊裝置跟網路媒體之間的溝通
# TCP/IP:
4  app  例如 HTTP DNS
3  transport  例如 TCP UDP
2  internet  網際網路協定
1  network assess  乙太 WIFI
